faq

Вопросы перед заказом

Это полноценный сайт?

Да. Это страница по ссылке, которую гость открывает с телефона. Внутри можно собрать всю важную информацию по событию.

Почему сейчас один пример?

Мы убрали разрозненные черновики из витрины и строим одну сильную систему. Так продукт будет выглядеть дороже и чище.

Можно сделать не свадьбу?

Да. После доведения Elegant-шаблона структура будет адаптироваться под день рождения, юбилей, корпоратив и другие события.

Что нужно для старта?

Дата, место, формат события, пожелания по стилю, список нужных блоков и фото при наличии.
