faq

Частые вопросы

Это просто картинка?

Нет. Это страница по ссылке: с датой, местом, программой, картой, RSVP, фото, текстом и QR-кодом.

Гости смогут открыть с телефона?

Да, сайт делается mobile-first. Основной сценарий — гость открывает ссылку в мессенджере с телефона.

Можно собрать ответы гостей?

Да. RSVP можно подключить через форму, чтобы ответы приходили в таблицу или на почту.

Можно срочно?

Если все данные готовы, можно сделать быстрее. Обычный срок — 3–5 дней.

Можно под любой праздник?

Да: свадьба, день рождения, юбилей, вечеринка, корпоратив, гендер-пати, выпускной, семейное событие.

Что нужно от клиента?

Дата, место, текст, стиль, фото при наличии, список нужных блоков и пожелания по оформлению.
