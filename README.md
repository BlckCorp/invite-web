# Invite Web

Лендинг для сервиса онлайн-приглашений и первый пример в портфолио.

## Что внутри

- `docs/index.html` — главный сайт Invite Web.
- `docs/portfolio/vladimir-birthday/index.html` — демо-кейс Premium-приглашения на день рождения.
- `docs/.nojekyll` — чтобы GitHub Pages отдавал файлы без обработки Jekyll.

## Как включить GitHub Pages

1. Открой репозиторий на GitHub.
2. Перейди в **Settings → Pages**.
3. В блоке **Build and deployment** выбери:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/docs**
4. Нажми **Save**.

После публикации сайт будет доступен примерно по адресу:

`https://blckcorp.github.io/invite-web/`

Портфолио-кейс:

`https://blckcorp.github.io/invite-web/portfolio/vladimir-birthday/`

## Важно

Демо-кейс сделан без публикации точного адреса. Для реального клиента личные данные и адреса лучше не выкладывать публично.
